# 🪶 R-package-update-check

> Навык, который автоматически собирает и описывает новые пакеты CRAN за последнюю неделю — и сохраняет результат в виде готового Markdown-дайджеста.

[English](README.md) | **Русский**

---

## Что делает

Каждую неделю на CRAN появляются сотни новых R-пакетов. R-package-update-check сканирует реестр, получает метаданные каждого нового пакета и формирует структурированный `.md`-дайджест с описаниями на понятном языке, категориями, зависимостями и прямыми ссылками.

**Один запрос → один Markdown-файл, готовый к публикации или отправке.**

---

## Быстрый старт

1. Установите навык в вашу среду ИИ-ассистента (см. [Установка](#установка))
2. Откройте диалог с ассистентом и напишите:

   > *«Покажи новые пакеты CRAN за эту неделю»*

3. Ассистент получит данные, сформирует описания и вернёт файл `cran-digest-YYYY-MM-DD.md` для скачивания.

---

## Формат вывода

Каждый дайджест имеет следующую структуру:

```
# Новые пакеты CRAN: 07.06.2025 — 13.06.2025

> Дайджест сформирован автоматически навыком R-package-update-check.
> Всего новых пакетов за период: 42

## Содержание по категориям
- Статистика: 8
- Визуализация: 6
- Машинное обучение: 11
- ...

## Пакеты

### awesome-pkg (0.1.0)
**Что делает:** ...
**Область применения:** Машинное обучение
**Зависимости:** torch, R6, cli
**Лицензия:** MIT
**Автор/мейнтейнер:** Jane Doe
**Ссылка:** https://cran.r-project.org/package=awesome-pkg
```

## Поддерживаемые параметры

Запрос можно уточнить на обычном языке — файлы конфигурации не нужны:

| Параметр | По умолчанию | Пример |
|---|---|---|
| Период | 7 дней | *«за последние 14 дней»* |
| Язык описаний | Язык запроса | *«descriptions in English»* |
| Фильтр по теме | нет | *«только пакеты для визуализации»* |
| Имя файла | `cran-digest-YYYY-MM-DD.md` | *«сохрани как weekly-cran.md»* |
| Макс. пакетов | все | *«топ-20 самых интересных»* |

## Установка

### Требования

- ИИ-ассистент с поддержкой навыков/агентов (например, Claude, OpenClaw, Hermes или любой совместимый рантайм)
- Файлы навыка из этого репозитория

### Шаги

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/NeonSalamander/r-package-update-check.git
   ```

2. Скопируйте `SKILL.md` (русский) или `SKILL-en.md` (английский) в директорию навыков вашего ассистента.

3. Перезапустите ассистента или перезагрузите навыки — навык появится в списке доступных автоматически.

## Источники данных

| Источник | URL | Назначение |
|---|---|---|
| CRAN по дате | `cran.r-project.org/web/packages/available_packages_by_date.html` | Список новых пакетов |
| crandb API | `crandb.r-pkg.org/{pkg}` | Метаданные пакета (основной) |
| CRAN DESCRIPTION | `cran.r-project.org/web/packages/{pkg}/DESCRIPTION` | Резервный источник |

## Поддержка языков

Файлы навыка доступны на двух языках. Язык описаний в дайджесте следует языку вашего запроса — дополнительная настройка не нужна.

| Файл | Язык навыка | Язык дайджеста |
|---|---|---|
| `SKILL.md` | Русский | Следует запросу |
| `SKILL-en.md` | Английский | Следует запросу |
